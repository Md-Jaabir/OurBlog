Introduction to programming

Programming is a method of giving instructions to a computer or electronic device. It involves creating a program, which is a set of instructions that the device follows to complete a task. The process of programming is akin to being a boss and your computer as your assistant, where you need to tell it what to do and how to do it.

To understand programming languages, there are three types of programming languages: machine language, low-level assembly language, and high-level structured programming languages. Machine language is a binary language that only a computer can understand. Initially, people only had machine language to program, which was difficult for programmers to remember the codes comprising only zeros and ones. Assembly language was invented to simplify programming to human beings, but it still required conversion into machine language so that a computer could understand it. Dennis Ritchie invented the first high-level programming language C in 1970, which takes two steps to compile. Firstly, it converts into assembly language, then the assembly code into machine code.

Choosing a programming language depends on the desired sector you want to work in. For example, if you want to be a web developer, you have to learn HTML, CSS, and JavaScript. If you want to be a mobile app developer, you have to learn Java or Kotlin for Android and Swift or Objective-C for iOS development. If you want to be a backend developer, you have so many options like Node Js (JavaScript runtime), Raw PHP, Laravel (PHP framework), Django (Python framework), Flask (Python framework), etc.

It is essential to choose a programming language based on your interests, career goals, and the sectors you want to work in. Learning a programming language is a continuous process, and it takes time, dedication, and practice to become proficient. The good news is that the internet is an excellent resource to learn programming, and there are several online courses, tutorials, and communities to help you get started.
